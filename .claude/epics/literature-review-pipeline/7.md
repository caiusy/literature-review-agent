# Task: 重写 synthesis-agent — 大规模支持 + 增强对比表

## Description

重写 .opencode/agents/synthesis-agent.md，支持 15-30 篇论文规模，增强对比表格维度，增加方法论趋势分析，研究建议扩展到 5-8 条。

## Acceptance Criteria

- [ ] 支持处理 15-30 篇笔记
- [ ] 分批读取策略：先扫描标题+方法做主题分组，再按组深入分析
- [ ] 对比表增加维度：Paper, Year, Method, Strengths, Weaknesses, Datasets, Key Metric, Limitations
- [ ] 增加 "方法论趋势" 小节
- [ ] 研究建议 5-8 条，每条引用具体论文
- [ ] 综述长度适配：15-20 篇 1500-2500 词，20-30 篇 2500-3500 词
- [ ] 输出到 outputs/review-overview.md

## Technical Details

分批处理策略：
1. 第一轮：扫描所有笔记，提取 title + research question + method
2. 第二轮：LLM 按主题聚类分组
3. 第三轮：逐组深入分析，生成组内对比
4. 第四轮：跨组综合，生成全局对比表和研究建议

输出结构：
```markdown
# Literature Review Overview
## Scope
## Thematic Summary
## Methodology Trends
## Comparison Table
## Cross-paper Analysis
## Research Suggestions
## References
```

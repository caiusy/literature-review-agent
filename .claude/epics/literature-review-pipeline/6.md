# Task: 重写 literature-reviewer — 读 Markdown 全文 + 提取笔记

## Description

重写 .opencode/agents/literature-reviewer.md，改为读取 data/papers-md/ 下的 Markdown 文件（而非 PDF），提取 9 字段结构化笔记。支持 abstract-only 模式和单篇指定路径。

## Acceptance Criteria

- [ ] 读取 data/papers-md/*.md（Markdown 全文，不截断）
- [ ] 提取 9 字段：title, authors, year/venue, research question, method, data/benchmark, key findings, limitations, relevance
- [ ] abstract-only 论文（*-abstract.md）基于摘要提取，标注 "Not available (abstract-only)"
- [ ] 支持 paper_path 参数指定单篇，便于并行调用
- [ ] 无参数时扫描 data/papers-md/ 全目录
- [ ] 笔记输出到 data/notes/{id}.md
- [ ] 笔记长度 200-500 词（比 V1 的 150-400 略增，适配更详细的 Markdown 输入）

## Technical Details

与 V1 的关键区别：
- 输入从 data/papers/ (PDF) 改为 data/papers-md/ (Markdown)
- 增加 abstract-only 检测（文件名含 -abstract）
- 增加 paper_path 参数支持
- 笔记模板不变，保持 9 字段结构

# Task: 配置 opencode.json 注册 MCP server

## Description

在项目根目录创建 opencode.json，将 MCP server 注册为 local 类型，使 OpenCode agent 可以调用 search_papers、download_paper、convert_pdf 三个 tool。

## Acceptance Criteria

- [ ] opencode.json 创建在项目根目录
- [ ] MCP server 配置为 local 类型，command 指向 python mcp-server/server.py
- [ ] OpenCode 重启后能识别 MCP server
- [ ] agent 可以调用 3 个 MCP tool

## Technical Details

```json
{
  "$schema": "https://opencode.ai/config.json",
  "mcp": {
    "literature-tools": {
      "type": "local",
      "command": ["python", "mcp-server/server.py"],
      "enabled": true
    }
  }
}
```

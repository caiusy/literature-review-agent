# Task: 重写 search-agent — 调 MCP tool + LLM 筛选

## Description

重写 .opencode/agents/search-agent.md，使其调用 MCP tool 完成检索和下载，LLM 只负责筛选评分。

## Acceptance Criteria

- [ ] agent 调用 search_papers MCP tool 获取候选列表
- [ ] LLM 对每篇论文进行四维度评分（相关性/引用量/年份/来源质量）
- [ ] 保留 Top 15-30 篇，输出 data/search-results/filtered.md
- [ ] 对筛选后论文逐篇调用 download_paper + convert_pdf
- [ ] 生成下载状态报告 data/search-results/fetch-status.md
- [ ] abstract-only 论文的摘要保存到 data/papers-md/{id}-abstract.md

## Technical Details

agent 工作流：
1. 接收用户关键词
2. 调用 MCP:search_papers(keywords, limit=30)
3. 对返回结果逐篇评分（LLM 推理）
4. 按总分排序，保留 Top N
5. 对每篇调用 MCP:download_paper → MCP:convert_pdf
6. 汇总状态报告

评分维度：
- 相关性（0-10，权重 40%）：LLM 判断标题+摘要与关键词匹配度
- 引用量（0-10，权重 20%）：>100=10, 50-100=7, 10-50=5, <10=3
- 年份（0-10，权重 20%）：当年=10, 每早一年 -1, 最低 2
- 来源质量（0-10，权重 20%）：顶会=10, 知名会议=7, 普通期刊=5, preprint=3

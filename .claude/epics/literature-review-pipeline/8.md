# Task: 编写 architecture.md + implementation-log.md

## Description

编写教程级别的系统文档。architecture.md 是系统架构的永久参考；implementation-log.md 是按时间顺序的实现过程记录，包含 CCPM 各阶段的说明。

## Acceptance Criteria

- [ ] docs/architecture.md 包含：
  - 系统架构图（ASCII 或 Mermaid）
  - 数据流说明
  - 每个组件的职责和接口
  - MCP server 的 3 个 tool 的参数和返回值
  - 3 个 agent 的输入/输出/触发方式
  - 目录结构说明
- [ ] docs/implementation-log.md 包含：
  - 每个任务的实现过程记录
  - 关键代码的逐行解释
  - 设计决策的理由（为什么选 marker-pdf、为什么用 MCP 等）
  - CCPM 各阶段（Plan/Structure/Sync/Execute/Track）的实际操作说明
  - 遇到的问题和解决方式
- [ ] 文档语言：中文为主，代码和技术术语保留英文
- [ ] 新人可据此文档理解整个系统

# Task: 创建 MCP server — search_papers, download_paper, convert_pdf

## Description

用 FastMCP 框架创建 Python MCP server，暴露 3 个 tool：search_papers（学术检索）、download_paper（PDF 下载）、convert_pdf（PDF→Markdown 转换）。

## Acceptance Criteria

- [ ] mcp-server/server.py 实现 3 个 tool
- [ ] search_papers：并行查询 Semantic Scholar + arXiv，去重，返回 JSON
- [ ] download_paper：下载 PDF 到指定目录，返回文件路径
- [ ] convert_pdf：用 marker-pdf 转 PDF→Markdown，返回 Markdown 路径
- [ ] server 可通过 `python mcp-server/server.py` 启动
- [ ] 每个 tool 有完整的参数校验和错误处理

## Technical Details

### search_papers
- Semantic Scholar: GET https://api.semanticscholar.org/graph/v1/paper/search?query={keywords}&limit={limit}&fields=title,authors,year,venue,citationCount,abstract,externalIds,openAccessPdf
- arXiv: GET https://export.arxiv.org/api/query?search_query=all:{keywords}&max_results={limit}&sortBy=relevance
- 去重：DOI 精确匹配 + 标题小写去标点后比较
- 返回统一格式 JSON

### download_paper
- 优先用 open_access_url
- arXiv 论文用 https://arxiv.org/pdf/{id}
- 2 秒间隔限流
- 文件名：{paper_id}.pdf

### convert_pdf
- 用 marker Python API：
```python
from marker.converters.pdf import PdfConverter
from marker.models import create_model_dict
from marker.output import text_from_rendered
```
- 输出 Markdown 到 data/papers-md/{stem}.md
